Provide a C program for an Operating Systems project that uses Cramer's Rule and pthreads to solve linear equations. This will serve as a benchmark comparison against a baseline multithreaded implementation.
Implement Gaussian elimination to support matrix sizes up to N=1000. The program must execute two performance tests back-to-back: a Thread Pool approach (where the number of threads equals the number of hardware CPUs) and a Thread-Per-Equation approach (where the number of threads equals N).
Additionally, the code must dynamically auto-detect the Linux username and hardware CPU count, append the final execution times to a solution.txt file, and suppress the printing of the calculated variable outputs if N > 10.
